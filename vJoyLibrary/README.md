# C# Library to access the vJoy Device
### Current Version: 1.1 B5

This is a supporting library for other projects.

Supports a C# interface for the vJoy virtual Joystick

Used to build game service applications such as SCJoyServer.

see: https://github.com/SCToolsfactory/SCJoyServer
